Welcome to PCU's documentation!
===============================

This documentation describes the PCU (Parallel C-Unit) testing framework,
distributed with and used by
`StGermain <http://www.stgermainproject.org>`_ -based applications.

If you're looking for information about System testing of StGermain-based apps,
check out the `CREDO documentation
<https://www.mcc.monash.edu.au/credo-doc/>`_.
