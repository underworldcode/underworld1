===============================================================================
BASIC STUFF
===============================================================================

-------------------------------------------------------------------------------
Grid

The "Grid" class represents a mapping from 1D integers to n-dimensional
parameterisations. It's useful in representing regular mesh vertex and element
indices.

-------------------------------------------------------------------------------
Decomp

Represents a decomposition of a set of indices. Can assist greatly in building
the set of local, remote and shared indices from the known set of indices
needed on a given process.

-------------------------------------------------------------------------------
Sync

Uses a "Decomp" object and adds information about the kind of data to be
synchronised. This way, individual arrays can be given to "Sync" objects for
easy synchronisation across processes.

-------------------------------------------------------------------------------
MeshTopology

Abstract class prescribing the required interfaces for a mesh topology.

-------------------------------------------------------------------------------
IGraph

Extension of "MeshTopology" using an incidence graph storage mechanism to
represent the topology of a mesh.

-------------------------------------------------------------------------------
MeshClass

Brings a "MeshTopology" together with a set of vertex coordinates, mesh
algorithms and element types.
